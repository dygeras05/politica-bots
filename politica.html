Política de Privacidade – Bot .gg/dygeras

Última atualização: 30/07/2025

1. Recolha de Dados

O bot .gg/dygeras não recolhe dados pessoais dos utilizadores.

As únicas informações processadas são:

  • IDs de utilizador, mensagens e servidores, quando necessário para o funcionamento do bot.
  • Estas informações não são guardadas de forma permanente.

2. Partilha de Dados

  • Nenhuma informação é partilhada com terceiros.
  • Não vendemos, alugamos ou cedemos dados a ninguém.

3. Armazenamento de Dados

  • O bot não mantém bases de dados permanentes.
  • Caso sejam guardadas informações temporárias para funcionamento, estas são eliminadas automaticamente.

4. Direitos dos Utilizadores

Se tiveres alguma dúvida sobre a utilização do bot, podes contactar o programador através de:

  • Discord: Dygeras.05
  • E-mail: dygeras05@gmail.com

Podes remover todos os dados temporários ao remover o bot do servidor.

5. Alterações a esta Política

Podemos atualizar esta política a qualquer momento.

Mudanças significativas serão comunicadas através da descrição do bot.
